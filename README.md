> A sleek, fully responsive personal portfolio built with **vanilla HTML, CSS & JavaScript** — featuring a custom animated preloader, scroll-reveal animations, text scramble effects, interactive project gallery, expandable experience section, and a floating pill navigation bar.

## ✨ Features

| Feature | Details |
|---|---|
| 🎬 Animated Preloader | Counting progress bar with smooth fade out |
| 🖱️ Custom Cursor | Lagging dot + magnetic button effects (desktop only) |
| 📜 Scroll Reveal | Smooth fade-up animations on section entry |
| 🔀 Text Scramble | Hero heading cycles through — Rohan / Designer / Builder / Founder |
| 🖼️ Project Gallery | Horizontal drag-to-scroll / mouse-wheel gallery |
| 📂 Experience Accordion | Click-to-expand job entries |
| 📊 Scroll Progress Bar | Thin accent-green line at the top of the page |
| 📱 Fully Responsive | Optimised for all screen sizes — 1440px down to 360px |
| 🧲 Magnetic Buttons | Social & send buttons follow the cursor on hover |
| 🃏 3D Tilt Cards | Skill cards tilt in perspective on hover |
